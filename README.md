# João Paulo - Personal Portfolio 🚀

> A personal portfolio website designed to showcase my projects, skills, and professional journey as a Full Stack and Mobile Developer.

## 📖 About The Project

This project is a modern, responsive, and interactive personal portfolio. It acts as a digital resume and a central hub for my recent works in Mobile Development, Web Apps, and Data & Automation. 

Built with vanilla web technologies, the portfolio features completely custom CSS styling, dynamic interactions, and a clean interface focused on highlighting the projects.

## ✨ Features

- **Modern UI/UX**: Sleek, glassmorphic design elements with smooth page transitions and micro-animations.
- **Fully Responsive**: Adapts seamlessly to all screen sizes, from mobile devices to large desktop monitors.
- **Dynamic Navigation**: Single-page application (SPA) feel using vanilla JavaScript for section switching without page reloads.
- **Project Categorization**: Projects are organized cleanly into distinct sections (Mobile, Web, and Data & Automation) using CSS Grid.
- **CV Download**: Integrated modal to download the resume in both Portuguese and English.
- **Sidebar Contact Info**: Quick access to social links and contact information, sticky on desktop for convenience.

## 🛠️ Technologies Used

The portfolio was built utilizing core web technologies to ensure lightweight and fast performance:
- **HTML5** (Semantics and Structure)
- **CSS3** (Custom Properties, Flexbox, CSS Grid, Keyframe Animations)
- **JavaScript** (Vanilla DOM manipulation, Event handling)
- **FontAwesome** (Icons)
- **Google Fonts** (Montserrat & Poppins typography)
